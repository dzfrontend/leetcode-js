# leetcode-js

leetcode真题js版，按照知识点抽取真题解析，为了方便验证结果，用jest来测试运行结果，真题代码在leetcode-js/code目录下。

## Examples

```bash
# 安装
$ npm install
# 运行jest测试查看示例结果
$ npm run test
```
